Detailed Description

wxArtProvider that provies PDF specific toolbar bitmaps

Call wxPDFViewArtProvider::Initialize() before using any of the art Ids.

  • wxART_PDFVIEW_PAGE_FIT Page fit bitmap
  • wxART_PDFVIEW_PAGE_WIDTH Page width bitmap
  • wxART_PDFVIEW_ZOOM_IN Zoom in bitmap
  • wxART_PDFVIEW_ZOOM_OUT Zoom out bitmap
